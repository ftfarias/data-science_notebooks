Course Schedule

Overview

The course will consist of a two-day intensive bootcamp before the start of the quarter, followed by a weekly seminar during the fall term where more advanced material will be discussed in-depth. The boot camp will be very hands-on and driven by short and long exercises where the concepts will be applied to programming puzzles as well as basic research problems.


Note: the following is still in draft form, and will likely be extensively modified before the course starts.

Much of the content below is adapted from the excellent resources made available by the Berkeley Python Bootcamp, as well as material on SciPy Lectures

    The homework assignment is to complete Parts I and II of the Breakout listed in the above IPython notebook. It will also give you practice working with Git and Github. For help on this, I recommend either going back through the lecture material above, or reading through http://gitref.org/.
    1. Create a new notebook called HW1.ipynb and in it complete Part 1 and 2 of the breakout. Use code cells for code, use heading and markdown cells for description, and make sure to execute the magic command %pylab inline to embed the resulting figures in the notebook.

    3. Create a new branch on your clone, and in that branch create a new directory that is your github username. Put your notebook in this directory, and use git commit to commit the result.
    4. Push the branch to your fork (git push origin branchname). Be aware that this will make your notebook public: anyone with web access can see what you wrote if they know where to look!
    5. From Github, submit a pull request to the original repository to turn-in your work.
